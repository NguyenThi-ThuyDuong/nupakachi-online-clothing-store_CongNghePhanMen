📌 Project Overview

NUPAKACHI Online Clothing Store is a web-based e-commerce system developed to simulate a real-world fashion retail platform.

The application enables customers to browse products, search items, manage shopping carts, and place orders. Administrators can manage products, categories, users, and order processing.

This project demonstrates the application of Software Engineering principles, including requirement analysis, system design, implementation, testing, and deployment.

🎯 Project Objectives

Apply Software Development Life Cycle (SDLC) methodology

Develop a complete MVC-based web application

Design and implement a relational database system

Practice teamwork and version control using Git

Build a functional e-commerce platform
